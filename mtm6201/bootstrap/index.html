Bootstrap This!

Bootstrap is a front-end framework for rapid development of Web applications and websites. Bootstrap is based on HTML, CSS, and JAVASCRIPT.


What is the benefits for Bootstrap?
Bootstrap is Easy to Use
Once you have the basics of HTML and CSS, you can start Bootstrap.
Browser support
All major browsers support Bootstrap.
Responsive design
Bootstrap's responsive CSS can adapt to desktops, tablets and mobile.
How to use Bootstrap?
Downloading Bootstrap
Bootstrap CDN
If you want to download and host Bootstrap yourself, go to getbootstrap.com , and follow the instructions there.
If you don't want to download and host Bootstrap yourself, you can include it from a CDN which is Content Delivery Network.
