HTML Tags

< hr >
< hr >is used to separate content in an HTML page.
< li >
< li > tag defines a list item.
< strong >
< strong > tag is used to define text with strong importance. The content inside is typically displayed in bold.
< select >
< select > element is used to create a drop-down list.
< section >
< section > tag defines a section in a document.
